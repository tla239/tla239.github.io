Power Sources Across the Nation

This map was built with custom JavaScript code drawing upon the Leaflet JavaScript Library. Although this map only displays three power sources: Nuclear, Hydro, and Wind, summary statistics are available for all power uses within a 500 km area. This map shows a comparison of the megawatt capacity of these power plants across the United States in two ways. First, there is a visual comparison using proportional symbols that are sized using a function with the megawatt capacity as an input. Secondly, the popups will display the megawatt capacity for each power plant in addition to the summary statistics. Users can turn on and off different power sources through the layer control window.

  • Data Source: U.S. Energy Information Administration

Find Your Distance to the Red Iguana Mexican Restaurant

This map was built with custom JavaScript code drawing upon the Leaflet JavaScript Library. The purpose of this map is to demonstrate how to calculate the distance from a user click to a known marker, in this case a Mexican Restaurant. When a new marker is created, the old one is removed upon each click from the user. The user is also able to drag the marker around on the map to a new location, and the distance is recalculated.

Colleges in Kansas

This map was built with custom JavaScript code drawing upon the Leaflet JavaScript Library. The intention of this map is to show how you can build a function to incorporate CSS styling rules into the popups for each University. Hence, each popup is built with a color specific to each school. In addition, the popup displays each school's logo with a clickable link to each University's website.
